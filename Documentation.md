# Project Documentation

## Task 04 – Extracting Rules from ML Models

### Phase 1: Research Phase (Pre-study)

#### Information Gathering

The first step of this project was to learn about rule extraction from machine learning models. I researched topics such as Explainable AI (XAI), Decision Trees, Rule Extraction, LIME, SHAP, and Surrogate Models.

I used different sources, including online articles, AI tools, and educational resources. I also used Generative AI to help me understand complex concepts and collect information more efficiently.

#### Tools and Technologies Chosen

For this project I used:

* Lovable
* Generative AI tools
* Web browser
* HTML, React and Tailwind CSS (generated by Lovable)

#### Project Planning

The project was divided into several stages:

1. Research rule extraction methods.
2. Learn how Explainable AI works.
3. Design a web application.
4. Create an interactive demo.
5. Add educational content and presentation slides.
6. Test the application.
7. Document and present the project.

#### Problems and Solutions

**Problem:** The topic was difficult to understand at first.

**Solution:** I used Generative AI to explain the concepts in simpler language and provide examples.

**Problem:** There are many different explainability methods.

**Solution:** I focused on the most common methods: Decision Trees, LIME, SHAP, Rule Extraction, and Surrogate Models.

## Phase 2: Implementation Phase

### Project Start

The project started by creating a web application called **Rule Extraction Playground** using Lovable.

The goal was to demonstrate how machine learning models can be explained through simple IF-THEN rules.

### Features Implemented

The application includes the following sections:

#### Demo

An interactive scatter plot displays red and blue data points.

A button called **Generate Rules** creates readable IF-THEN rules such as:

IF x > 5 AND y < 3 THEN Class = Blue

This demonstrates how a complex model can be translated into simple human-readable rules.

#### Story

This section explains what rule extraction is and why it is important for understanding machine learning models.

#### Methods

The Methods section describes:

* Decision Trees
* Rule Extraction
* Surrogate Models
* LIME
* SHAP

Each method includes explanations and examples.

#### Slides

A presentation mode was added with educational slides explaining the project topic.

#### Cheat Sheet

A comparison table summarizes the different methods and their purposes.

#### GenAI

A dedicated section explains how Generative AI was used throughout the project.
