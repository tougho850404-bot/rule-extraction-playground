# Rule Extraction Playground

## Project Overview

This project demonstrates how machine learning models can be explained using rule extraction techniques.

The application was created using Generative AI and Lovable.

## Features

- Interactive Demo
- Rule Generation
- Story Section
- Methods Section
- Slides
- Cheat Sheet
- GenAI Explanation

## Methods Covered

- Decision Trees
- Rule Extraction
- Surrogate Models
- LIME
- SHAP

## Technologies

- Lovable
- React
- Tailwind CSS
- Generative AI
